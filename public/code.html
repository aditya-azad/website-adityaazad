MindsDB

I contributed to MindsDB during Hactoberfest 2023! I integrated PyCaret, NPM, SAP ERP, Lightdash, Apache Doris, Xata, Milvus and Pinecone with MindsDB during the month of October. It was an incredible learning experience since I got to know how open source works in a for profit company, and improve all kinds of software adjacent skills at the same time. It was like a mini internship where I got to provide a lot of value in so little time.

Personal Projects

Simple Static Site Generator

This was my first stab at a transpiler and an exercise in simplicity. All the latest and greatest static site generators are either too bloated or have complex language and mechanisms to maximize flexibility. Chances are, if you are using a static site generator, you are building something simple. Why make the generator so complex? It is the builder used for this site. I hope to write a more robust and faster version in future.

Destined

A minimal task management software I built when I first learned React in the good old days. What makes this different is various color coding schemes, sorting mechanisms and repeatable tasks. I kept the code dependencies to a minimum to get a feel for the JavaScript ecosystem and the "lovely" Webpack configuration system. I don't use this anymore because of the unnecessarily complexity that I thought was necessary at the time.

Candle Grey

This is a colorscheme I designed for Vim8. It was an exercise in contrast and values (color theory). I wanted to make it as simple as possible and see if I can get away with virtually no eye candy. I also wanted to force myself to write better code. Taking away the crutches of bright colors forced me to design the code in a pleasing, readable way.
