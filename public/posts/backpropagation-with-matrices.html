Backpropagation With Matrices

2023-07-30
machine-learning,

There are a lot of online resources that show how backpropagation works on scalar values1 (probably because it's easy to understand), but there are not enough of them talking about doing it using matrices. So, in this article I will built it up step by step on a single dense layer.

Forward pass

First, let's go through forward pass using a single neuron. Each neuron has a set of inputs which it receives from previous layer. We collect these in a vector and name that vector x. For each of these inputs, we have weights associated with it.

def fib(x):
    if x == 1:
        return 1
    elif x == 2:
        return 1
    p1 = p2 = 1
    for _ in range(x-2):
        temp = p1 + p2
        p1 = p2
        p2 = temp
    return p2
